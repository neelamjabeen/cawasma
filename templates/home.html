OWASP Top 10 — 2025

Security Misconfiguration is the #2 most critical web risk

Existing black-box scanners detect misconfigurations but lack meaningful prioritization. CAWASMA bridges this gap with context-aware CVSS scoring — considering endpoint sensitivity, response data, and attack chain correlation.

Core Capabilities

What CAWASMA Detects

HTTP Header Analysis

Checks Content-Security-Policy, HSTS, X-Frame-Options, Referrer-Policy, and more against best-practice configurations.

CORS Misconfiguration

Identifies wildcard origins, insecure preflight handling, and overly permissive cross-origin resource sharing policies.

TLS / SSL Configuration

Validates cipher suites, protocol versions, certificate validity, and HSTS deployment across all discovered endpoints.

NLP Endpoint Classification

Uses natural language processing to identify endpoint types (admin, login, payment) and adjusts risk scoring based on sensitivity.

Attack Chain Correlation

Maps multi-step exploit paths by correlating individual findings into broader attack chains with aggregated risk scores.

Multi-Format Reports

Generates detailed findings reports in PDF, JSON, HTML, and CSV — ready for developers, auditors, and stakeholders.

Workflow

How It Works

A fully passive, black-box analysis pipeline — no source code access, no intrusive probing, safe for production targets.

1
Enter Target URL
Provide the target domain or API base URL. CAWASMA operates entirely over HTTPS with no internal access required.
2
Automated Endpoint Discovery
Crawls HTML, parses JavaScript, and probes common paths to build a comprehensive endpoint map.
3
Context-Aware Analysis
Each endpoint is classified by type and sensitivity. Detection modules run against headers, cookies, methods, and response data.
4
CVSS Scoring & Prioritization
Findings are scored with context-adjusted CVSS metrics — prioritized by actual risk, not just presence of a misconfiguration.
5
Report Generation
Download structured reports in your preferred format (PDF, JSON, HTML, CSV) with remediation guidance.
